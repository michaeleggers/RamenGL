# Some useful sites to get asset files from. (Check the license)

- PBR Textures
  - [ambientcg](https://ambientcg.com/)

- Skyboxes
  - [humus](https://www.humus.name/index.php?page=Textures) 

- Models
  - [Sketchfab Creative Commons](https://sketchfab.com/3d-models?date=week&features=downloadable&licenses=322a749bcfa841b29dff1e8a1bb74b0b&licenses=b9ddc40b93e34cdca1fc152f39b9f375&licenses=72360ff1740d419791934298b8b6d270&licenses=bbfe3f7dbcdd4122b966b85b9786a989&licenses=2628dbe5140a4e9592126c8df566c0b7&licenses=34b725081a6a4184957efaec2cb84ed3&licenses=7c23a1ba438d4306920229c12afcb5f9&sort_by=-likeCount&cursor=bz0xJnA9MjU%3D) (free account needed in order to download.)
  - [Quaternius.com](https://quaternius.com/index) Complete assetpacks (often with blend files) with permissive license.
  - [The models resource](https://models.spriters-resource.com/) Assetfiles extracted from Videogames.
